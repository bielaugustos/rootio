# Neobrutalism Design - Implementation Plan

## Overview
Transform Rootio web app into a stunning neobrutalism design with bold borders, vibrant colors, and hard offset shadows.

---

## Current Issue - Dark Mode Not Working Properly

The current dark theme has these problems:
1. Dark mode colors don't change - background remains light
2. Border color stays as black instead of switching to white
3. Shadows remain black instead of switching to white
4. No proper dark neobrutalism aesthetic

### Required Dark Mode Fix:
- Dark background: `#0D0D0D` (near black)
- Surface: `#1A1A1A` (dark gray)
- White (text/elements): `#FFFFFF` (pure white)
- Ink (text): `#FFFFFF` (white for dark mode)
- Border: `#FFFFFF` (white borders - key neobrutalism feature)
- Shadow: `#FFFFFF` (white shadows for dark mode)
- Keep vibrant accent colors but adjust for dark backgrounds

---

### Phase 1: Fix Dark Mode Colors

Update the `[data-theme='dark']` section with proper neobrutalism dark mode:

### Update Neobrutalism Specs for Dark Mode

Add CSS that uses the correct black/white values based on theme:
- Light mode: black borders, black shadows, light background
